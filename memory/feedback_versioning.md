코드 수정 시 반드시 앱 버전을 올린다. 꼬마가 별도로 요청하지 않아도 무조건 적용.

- 대규모 수정 (새 단계 완성, 아키텍처 변경, 대규모 기능 추가 등) → **+1** (예: 2.0.0 → 3.0.0)
- 중간 규모 수정 (새 기능 추가, 여러 파일 변경 등) → **+0.1** (예: 2.1.0 → 2.2.0)
- 사소한 수정 (CSS 미세 조정, 버그 1건 수정 등) → **+0.01** (예: 2.1.0 → 2.1.1)

**Why:** 꼬마가 40번 넘게 수정했는데 버전이 1.0.0에 머물러 있어서 지적함. 변경 이력을 버전으로 추적하고 싶어함.

**How to apply:** `src/constants/appVersion.js`, `package.json`, `public/sw.js` 세 곳 동시 변경.
